## Recording Tips

```
1. Record ALL scenes in one session (edit later)
2. Pause 3 seconds between scenes (easier to cut)
3. Click slowly and deliberately
4. Keep mouse movements smooth
5. If you make a mistake, pause, restart that scene
6. Record voice separately if easier (sync in CapCut)
7. Speed up typing in editing (nobody wants to watch typing)
```

## Post-Recording Edit Checklist

```
□ Cut loading times
□ Speed up typing (2x-3x)
□ Add text overlays for key moments
□ Add subtle zoom on important clicks
□ Add background music (15-20% volume)
□ Add captions (CapCut auto-generates)
□ Add logo + URL at end
□ Color correct if needed
□ Export at 1080p
□ Create thumbnail
```
